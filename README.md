# BuildCompresserFumino

一个便利性插件，可以将同种可放置物品压缩为单个物品。

## 功能简介

- 同种可放置物品压缩
- 直接获取指定压缩量的物品，需要管理员权限
- 压缩物品放置逻辑与原版相同

## 依赖

| 依赖 | 说明 |
|------|------|
| Paper 1.21+ | 服务端核心 |

## 指令

| 指令 | 权限 | 描述 |
|------|------|------|
| `/compress` | 无 | 打开物品压缩界面 |
| `/givecompressed` | 需要管理员权限 | 给予玩家指定数量的指定压缩物品 |
